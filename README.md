# MySqlDocFx
数据库文档生成图形化工具

Auto create mysql doc to markdown or html

##### 工具使用：
以下两种方式都可以使用本工具
- 直接下载生成好的jar包
运行以下命令：

```
java -jar mysqldocfx-1.0-SNAPSHOT-jar-with-dependencies.jar
```


- 自行编译打包 
```
 mvn package
```
