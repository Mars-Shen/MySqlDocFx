# MySqlDocFx
一个数据库表结构自动生成Markdown或者Html文档的图形化工具

A tool for auto create mysql doc to markdown or html with UI

##### 工具使用：
以下两种方式都可以使用本工具
- 直接下载生成好的jar包
运行以下命令：

```
java -jar mysqldocfx-1.0-SNAPSHOT-jar-with-dependencies.jar
```


- 自行编译打包 
```
 mvn package
```
填写配置后点击生成, 即可自动生成数据库文档
